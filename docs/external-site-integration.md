# External Site Tracking Integration

The dashboard accepts public tracking events at `/api/track` on the dashboard deployment, for example:

```txt
https://abl-dashboard.netlify.app/api/track
```

When testing from inside the dashboard app, `/api/track` also works through the Netlify redirect.

## Required Dashboard Environment

Set `TRACKING_ALLOWED_ORIGINS` in the dashboard deployment to include both public site origins:

```bash
TRACKING_ALLOWED_ORIGINS=https://quantum-frequency-assessment.netlify.app,https://timewaver-audio-sales.netlify.app
```

The ingest API validates the browser `Origin`, `site_id`, event name, and `page_url` host. A public site must send events from its own origin with the matching site ID.

## Quantum Frequency Assessment

Use this site ID:

```txt
quantum_frequency_assessment
```

Allowed conversion events:

```txt
assessment_submit
line_click
consultation_booking
```

Initialize page views once on page load by pasting the inline snippet generated for:

```ts
trackingSnippet("quantum_frequency_assessment", "https://abl-dashboard.netlify.app/api/track");
```

The generated snippet creates `window.ablTracker` and immediately calls `window.ablTracker.pageView()`.

Track conversions after successful user actions:

```html
<script>
  window.ablTracker.track("assessment_submit", { form: "main-assessment" });
  window.ablTracker.track("line_click", { label: "line-consultation" });
  window.ablTracker.track("consultation_booking", { form: "consultation" });
</script>
```

## TimeWaver Audio Sales

Use this site ID:

```txt
timewaver_audio_sales
```

Allowed conversion events:

```txt
audio_purchase_click
line_click
payment_success
```

Initialize page views once on page load by pasting the inline snippet generated for:

```ts
trackingSnippet("timewaver_audio_sales", "https://abl-dashboard.netlify.app/api/track");
```

The generated snippet creates `window.ablTracker` and immediately calls `window.ablTracker.pageView()`.

Track conversions after successful user actions:

```html
<script>
  window.ablTracker.track("audio_purchase_click", { product: "timewaver-audio" });
  window.ablTracker.track("line_click", { label: "line-sales" });
  window.ablTracker.track("payment_success", { product: "timewaver-audio" });
</script>
```

## Inline Paste Snippet

The dashboard app does not currently build a public `/assets/abl-tracker.js` file for other sites to import. For plain external sites, use the inline snippet generated by `trackingSnippet(siteId, endpoint)` from `src/tracking/snippet.ts`, then paste the returned `<script>` tag into the public site.

## Metadata Privacy Rule

Do not send names, phone numbers, email addresses, payment card data, private form answers, or any other personally identifiable information in `metadata`.

Use short business-analysis labels only, such as:

```js
window.ablTracker.track("line_click", { label: "hero-line-button" });
window.ablTracker.track("payment_success", { product: "timewaver-audio", amount: 1880 });
```

The browser helper keeps only flat primitive metadata values and drops obvious private keys. The server also normalizes non-object metadata to `{}`.
